# Culturefork - review part

> 각 지역의 다양한 문화 행사 및 이벤트 정보를 제공하고, 사용자 간 후기를 공유할 수 있는 문화 종합 커뮤니티 서비스
>
> * 담당기능 : 문화 관람 후기 게시판


##### 1. 사용기술

* 웹 표준
  * `HTML5`
  * `CSS3`
  * `JavaScript`
  * `jQuery`
  * `Bootstrap`

* DMBS
  * `MySQL`
* `Spring Framework`
* `MyBatis`
* `jQuery`, `Ajax`, `XML`, `JSON 파싱`
* `REST 인터페이스` 구현

* API
  * `공공데이터 포털 오픈 API - 공연 전시 정보 조회 서비스`
  * `카카오 Map API`
